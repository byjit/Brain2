# Getting Started with TanStack Router

TanStack Router is a fully type-safe router for React. It features built-in search params validation, loader functions, code-splitting, and first-class integration with TanStack Query.

## Key Concepts

* **Type Safety**: Your routes, search params, and loaders are completely type-safe.
* **Nested Routing**: Nest pages easily with layouts and outlet components.
* **Search Param APIs**: Fully validate and statefully manage URL search parameters.
* **Pending / Error States**: Handle loading and error boundaries out-of-the-box.

### Simple Route Example

Here is how you register a basic route:

```typescript
import { createFileRoute } from '@tanstack/react-router'

export const Route = createFileRoute('/about')({
  component: AboutComponent,
})

function AboutComponent() {
  return <div>About Us</div>
}
```

Try navigating to this and other pages using the links in the header!
