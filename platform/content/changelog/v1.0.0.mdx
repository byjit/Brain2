# Initial Release (v1.0.0)

We are thrilled to announce the v1.0.0 release of our premium React TanStack Boilerplate.

## What's Included?
- **TanStack Router**: File-based, fully type-safe routing out of the box.
- **TanStack Query**: High-performance caching, fetching, and synchronization of server state.
- **Tailwind CSS v4**: Blazing fast utility-first styling with modern browser standards.
- **Zustand**: Clean and lightweight state management for global UI state.
- **Content Collections**: Seamless, type-safe markdown/MDX integration for blogs and pages.
- **Biome**: Next-generation linter and formatter.
